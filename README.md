comandos para a base de dados 
```
php artisan migrate
php artisan migrate:refresh
php artisan migrate --path=/database/migrations/2024_11_27_094821_create_reports_table.php
```

criar ficheiros 
```
php artisan make: Nome_da_pasta nome_do_ficheiro
```

Mandar explos para  a base de dados 
```
php artisan db:seed     
```

```
  php artisan serve
```
